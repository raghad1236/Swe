PrepWise — Web Application

-Project Overview:

PrepWise is a web application designed to help users prepare efficiently for exams and quizzes by providing personalized dashboards, progress tracking, and study schedule. The platform focuses on usability and clean navigation to enhance the learning experience.

-Key Features:

User Authentication – Log in and sign up functionality for personalized access.

Dashboard – Displays user's progress, and upcoming tasks.

Plan page – clear and personalized weekly schedule.

Profile Page – View and edit user details.
Achievement Page-View earned badges and shows motivational popups when achievements are unlocked
Responsive Design – Works seamlessly across devices.

-Interface Design

The interface was designed following UX guidelines such as consistency, clarity, feedback, and accessibility. A user navigation hierarchy diagram is provided to show the page flow and structure of the application.

-Technologies Used

Frontend: HTML, CSS, JavaScript

Tools: GitHub, Jira
